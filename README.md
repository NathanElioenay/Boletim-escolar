# 📚 Boletim Escolar

Sistema simples de gerenciamento de alunos desenvolvido em Python para praticar conceitos fundamentais da linguagem.

## Funcionalidades

* Cadastro de alunos
* Cálculo automático da média
* Listagem de alunos
* Listagem em ordem alfabética
* Exibição do aluno com maior média
* Cálculo da média geral da turma
* Remoção de alunos
* Salvamento automático dos dados em arquivo `.txt`

## Tecnologias Utilizadas

* Python 3

## Conceitos Praticados

* Estruturas condicionais (`if`, `elif`, `else`)
* Estruturas de repetição (`while`, `for`)
* Manipulação de listas
* Leitura e escrita de arquivos
* Tratamento de exceções (`try` / `except`)
* Entrada e saída de dados
* Organização de menus interativos

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/NathanElioenay/boletim-escolar.git
```

2. Acesse a pasta do projeto:

```bash
cd boletim-escolar
```

3. Execute o programa:

```bash
python boletimescolar.py
```

```text
======= BOLETIM ESCOLAR =======

1 - Cadastrar aluno
```

## Objetivo
